To generate 1-d bifurcation diagram
1. In XPP, (I)(G) go, then (I)(L) Last SEVERAL TIMES! (like twice)
2. (F)(A) Open AUTO, set parameter, set axes to hI-lo, set Numerics: Ds to be positive first.
3. Run
if succeed
4. Grab initial point (point 1)
5. change Ds in Numerics to negative
6. Run

To get eigenvalues
after the above 3.
4. Grab the point you want
5. click XPP window, (I)(G), (I)(L)
6. click Sing pts in the left column
7. click (G)o
8. Print eigenvalues? YES
9. then the eigenvalues will be printed in the cmd
